# Рекомендательная система по премиум-танкам для игры Tanks Blitz

**Состав разработчиков:** индивидуальный проект.

**Цель проекта:** Индивидуализация предложений на приобретение премиум-техники в игре Tanks Blitz (текущая система предполагает одинаковое предложение для всех игроков на определенный период времени без учета наличия данной техники в ангаре у игрока).

**Оборудование:** персональное.

**Данные:** собранная по [API](https://developers.lesta.ru/) информация о среднем уроне игроков на определенной технике.

**Риски:**
1. Не уложиться в сроки.
2. Недостаточное количество данных для выявления закономерностей.
3. Сложные закономерности или их отсутствие, из-за чего результаты неудовлетворительны.
4. Наличие ошибок и шума в данных, из-за чего результаты неудовлетворительны.
5. Изменение API или его закрытие, что приведет к блокировке получения данных.
6. Недостаточная мощность персонального оборудования для разработки модели.

**Метрика оценки результатов моделирования:** $R^2$ (минимальный порог – 0.2, оптимальный – 0.5)

**План проекта:**
1. Сбор данных посредством API (24.10-02.11).
2. Анализ полученных данных и их подготовка (03.11-11.11).
3. Обучение модели и оценка ее качества (12.11-07.12).
4. Создание API обращения к модели (08.12-19.12).
